# SmartWardrobe

SmartWardrobe is a web application that helps users manage outfits and clothing items in a smart and organized way.

## Features
- User authentication (login & register)
- Add, edit and delete clothing items
- Outfit recommendations based on weather and from the pictures uploaded
- Responsive UI
- Backend API for data management
- Weather based on location


  ## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: PostgreSQL
- Tools: npm

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Mpp77/SmartWardrobe.git
```
2. Install dependencies:
```bash
npm install
```
3. Configure environment variables:
Create a .env file and add:
```bash
DATABASE_URL=postgresql://user:password@localhost:5432/smartwardrobe
```
Run the application:
```bash
npm start
```
